# Catalog Page — HTML/CSS (Semantics + Grid)

Учебный проект: страница каталога товаров без JavaScript.  
Фокус — семантика, архитектура разметки, CSS Grid, адаптивная сетка карточек.

## Цели проекта
- Понять семантику каталога: где `section`, где `article`, где `aside`
- Развести архитектуру страницы и контентные компоненты
- Сверстать сетку каталога и карточек на **CSS Grid**
- Сделать адаптив под мобильные/планшет/десктоп
- Отработать позиционирование (бейджи поверх карточек)

## Что реализовано
- Семантическая структура: `main`, секции каталога, `aside` для фильтров, карточки как `article`
- Двухколоночный layout: фильтры + результаты (`280px 1fr`)
- Сетка карточек: `repeat(auto-fit, minmax(...))`
- Карточка на Grid: паттерн `auto 1fr auto` (контент + “кнопка снизу”)
- Бейджи/иконки поверх карточки через `position: relative/absolute`
- Адаптив:
  - на мобильных фильтры становятся сверху
  - карточки перестраиваются в 1 колонку на узких экранах
  - корректные отступы контейнера на mobile

## Технологии
- HTML5 (семантика)
- CSS3 (Grid, flex локально внутри компонентов, media queries)
- Без JavaScript, без библиотек
